# 🌸 Musubi

> *Musubi (結び) — the Japanese art of connection. A knot that binds two things together.*

Musubi is a Discord bot that connects servers to each other in real time — like a telephone network built into Discord. Two communities dial in, and for the duration of the call, their members can talk to each other through a shared relay channel called the **booth**. When the call ends, the line goes quiet. No permanent bridges, no cross-server memberships — just a moment of genuine connection between strangers.

The vision behind Musubi is simple: **Discord communities are too isolated.** Servers grow inward, develop their own cultures, and rarely encounter anyone new. Musubi adds a layer on top of that — a network that lets communities reach outward, meet strangers, share a conversation, and maybe make a friend.

## ✨ Features

### 📞 Call System
- `/call` — dial into the network and connect with a random server
- `/hangup` — end your call or cancel an active search
- Calls auto-end after 30 minutes of inactivity
- Search times out after 30 seconds if no match is found

### 🎭 Identity & Anonymity
- `/anonymous` — hide your name and avatar; appear as `📞 Anonymous`
- `/friendme` — share your Discord tag with the other server mid-call
- `/me name` — set a custom relay display name *(User Premium)*
- `/me avatar` — set a custom relay avatar *(User Premium)*
- `/me reset` — revert to your real Discord identity

### 🛡️ Safety & Filtering
- Global blocklist for words and phrases
- Invite link blocking, caps filter, flood detection, and repeat-message detection
- Per-user and per-guild banning from the network
- All filtering is silent — no notifications to the sender

### 🏆 XP & Callboard
- Servers earn XP for every message sent during a call
- `/callboard` — view the top 7 most active servers this cycle
- XP resets monthly; past cycles are saved for history
- Guild Premium doubles XP earnings

### ✨ Premium
- **User Premium** — custom nickname and avatar during calls
- **Guild Premium** — file and attachment relay, sticker relay, double XP, priority matchmaking
- Keys generated by admins and redeemed with `/redeem`

### ⚙️ Server Setup
- `/setup` — register your server and set a booth channel
- `/setbooth` — change the booth channel at any time
- `/unregister` — remove your server from the network
- `/prefix server` — set a custom command prefix

### 🔧 Network Administration
- Tiered permissions: **owner** → **sudo** → **user**
- Sudo users can ban members and guilds, manage premium, generate keys, terminate calls, and broadcast notices across the network
- Live cog reloading without restarting the bot

## 🚀 Getting Started

### Prerequisites
- Python 3.11+
- A Discord bot token with `message content` and `members` intents enabled
- A Supabase project (free tier works fine)

### Environment Variables

```
DISCORD_TOKEN=your_bot_token
OWNER_ID=your_discord_user_id
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your_supabase_anon_or_service_key
```

### Installation

```bash
git clone https://github.com/yourname/musubi.git
cd musubi
pip install -r requirements.txt
python main.py
```

### Database

Apply the schema in `supabase_database_schema.sql` to your Supabase project. A monthly cron job should be configured in Supabase to snapshot `Guilds` XP into the `Leaderboard` table and reset `Guilds.xp` to `0`.

## 📁 Project Structure

```
musubi/
├── main.py            # Bot entrypoint, prefix resolution, error handling
├── botprotocol.py     # Typed bot protocol for Pyright compatibility
├── datamanager.py     # All Supabase I/O, caching, and business logic
├── embeds.py          # Centralised embed factory
├── flank.py           # Lightweight keep-alive HTTP server (Render free tier)
└── cogs/
    ├── phone.py       # /call, /hangup, /anonymous, /friendme
    ├── bridge.py      # Message relay, XP batching, delete sync
    ├── premium.py     # /me, /premium, /redeem
    ├── sudo.py        # /sudo (admin commands)
    ├── filter.py      # /filter + message filtering logic
    ├── leaderboard.py # /callboard
    └── help.py        # /help
```

## 🤝 Contributing

Musubi started as a personal project but the vision is bigger than one person. If you believe Discord communities deserve a way to reach beyond their own walls — and you want to help build the network that makes that possible — you're welcome here.

There's no corporate roadmap, no ticket queue, no bureaucracy. Just a bot with a clear purpose and room to grow.

**Ways to get involved:**
- 🐛 Found a bug? Open an issue and describe what happened
- 💡 Have a feature idea? Open an issue and pitch it — if it fits the vision, let's build it
- 🔧 Want to write code? Fork the repo, make your changes, and open a pull request
- 📖 Good at writing? Help improve the docs, help system, or in-bot messaging
- 🌐 Running a community? Add Musubi to your server and give us real-world feedback

If you're unsure whether your idea fits, just open an issue or jump into the Discord and talk it through. The only bar is that it serves the core mission — bringing communities together.
